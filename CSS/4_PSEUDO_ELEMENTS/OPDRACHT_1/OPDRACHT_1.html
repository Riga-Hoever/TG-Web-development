TERUG NAAR ANTWOORDEN

1) Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)?
Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

  • ::after
    •  Plaatst content na een aangegeven element.
  • ::before
    •  Plaatst content voor een aangegeven element
  • ::first-letter
    •  Selecteerd de eerste letter van elk aangegeven element
  • ::first-line
    •  Selecteerd de eerste lijn van een aangegeven element
