1) Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen?
Leg ook voor elke wijze uit wat het precies inhoud.

Er zijn 3 verchillende manieren om CSS in te laden:

  1. Inline CSS
  2. Internal CSS
  3. External CSS
1.Bij inline CSS wordt de STYLE element direct toegepast op de tag die nodig is. Zie bron code voor voorbeeld! Verder heeft de inline CSS
voorang bij het bewerken van een tag. 2.Voor Internal CSS wordt de STYLE element gebruikt in de HEAD gedeelte van een HTML document. Zie de STYLE element in de HEAD sectie @ de bron voor
een voorbeeld!
3. Met external CSS wordt er een externe document gebruikt om alle design aspecten aan te geven van een HTML pagina. Deze
wordt via een link verbonden in de head element. Zie comment in de bron in de head tag voor uitleg!
