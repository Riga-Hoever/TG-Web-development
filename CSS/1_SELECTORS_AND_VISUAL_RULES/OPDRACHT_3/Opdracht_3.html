3) Er zin verschillende soorten selectors, bijvoorbeeld:
  1. Type selectors
  2. Decendant selectors
  3. Class selectors
Maak van de bovenbenoemde selectors voorbeelden. Geef in elk antwoord ook aan wat de voordelen zijn voor elke selector.

Met behulp van de Type selectors geef je een specifiek node element een waarde door het gehele document. Bijvoorbeeld
als je de h1 een waarde geeft, dan krijgt elke h1 element in het docuent dezelfde waarde die is aangegeven met behulp van de Type selector.

De Decendant selectors passen de elementen aan die zich bevinden in een ander element. Een voorbeeld daarvan is bijvoorbeeld de h1
element die zich op dat moment in een div zit. Dan pakt css allen h1 in combinatie met een div aan.

Class selectors pakt alle namen aan die worden gegeven in de html document. Bij bijvoorbeeld het geven van een naam aan een div met
behulp van de class selector dan pakt het alle elementen aan die zich in die division bevinden.
