2) CSS pas je toe met een bepaalde Syntax. Hoe ziet zo'n syntax eruit? kun je ook uitleggen wat elk deel uit de syntax betekent?

Een CSS Syntax bestaat uit een Selector en een Decleration. De selector betreft de element die aangepast moet worden, en de declaratie bestaat
uit een Css property met een naam, en het andere deel een waarde die de property krijgt. Zie het voorbeeld hieronder.
