Cascade

Cascade is een concept die je kan zien als een waterval. Als een pagina meerderen elementen bezit die hetzelfde zijn
en je vervolgens bijvoorbeeld een paar span elementen erin gooit maar ze hebben alemaal verschillende kleuren als property, dan zal de code alle span elementen langsgaan van boven naar beneden en de laatste span die erin staat aanwijzen als een property. Het zal eerst langs de html bron de code langsgaan, komt vervolgens indien gemaakt eerst langs de link naar de css stylesheet en gaat vervolgens
daarna verder met de rest van het html document. Als er 3 color elementen voorkomen in het css bestand, en er vervolgens een staat onderaan het html
bestand, zal deze worden aangewezen omdat het de laatste is van de reeks. Net als een warerval die zijn weg volgt.

inherratence

De inheritance tag zorgt ervoor dat meerderen tags hetzelfde styling krijgen als ze de child + grandchild zijn van een tag.
