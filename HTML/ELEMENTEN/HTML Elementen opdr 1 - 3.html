Vraag 1

Bij het inladen van een bestand is het noodzakelijk om de locatie van het bestand aan te geven.Dit kan op 2 manieren:
  • Absolute path
  • Relative path

    • Wat houden deze 2 paths in?

    Absolute paths hebben altijd de gehele link staan van het domein waar het heen moet leiden. Terwijl Relative links
    alleen wijzen naar een bestanden of pad naar een bestand toe.

    Vraag 2

    Zoek uit welke formaat afbeeldingen er nog meer kan zijn.

    Er zijn zowel met afbeeldingen pixel als vector afbeeldingen. Pixel heeft een fixed formaat, terwijl een vector formaat oneindig
    uitgerekt kan worden zonder dat de resolutie slechter wordt. Een pixel bestand daarintegen, zal dermate je de foto bromeert groter maakt,
    zijn kwaliteit verliezen. Hieronder staan enkele voorbeelden van afbeelding bestands types die het meest worden gebruikt op internet, maar er zijn er nog veel meer.

    JPEG : Pixel bestand
    PNG Pixel bestand
    GIF ( Pixel bestand heeft minder kleuren dan de rest van de bestands types)
    TIF Pixel bestand
    EPS + AI Vector bestand ( Wordt meestal gebruikt voor bijvoorbeeld logo's )
