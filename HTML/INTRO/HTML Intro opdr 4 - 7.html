Vraag 6

Waarom is een goede HTML structuur belangrijk? is deze code een goede structuur? leg uit.

Nee. Want ook al ziet het voorbeeld er gestructureerd uit, er worden geen gebruik gemaakt van tags.
Dus die mooie lap tekst die mooi verdeeld staat wordt 1 brok tekst op de webpagina.
Een goede HTML structuur is belangrijk voor het overzicht. Als een pagina veel informatie bevat wil je niet
eindeloos zoeken naar de juiste stukken code. Plus er bestaat een grote kans dat een ander persoon jouw code moet bewerken.
Dus overzicht is dan voor diegene beangrijk en zeer tijd besparend!
