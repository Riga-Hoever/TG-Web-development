Vraag 18

Onderzoek welke 4 attributen een link tag kan hebben. Leg uit wat deze doen en voeg deze toe als voorbeeld op je website.

De download attribute laat zien aan een gebruiker waneer een file wordt gedownload als er wordt gedrukt op de hyperlink.
De Href attrubite laat de url zien van pagina waar de link heen gaat.
De Rel attribtute laat de relatie zien tussen een huidig document en de gelinkte document.
